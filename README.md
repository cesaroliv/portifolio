# Portfólio — Estúdio de Criativos com IA

Site de portfólio de uma página (HTML, CSS e JavaScript puro) para captar clientes.
Conta o case da rosa mosqueta: de uma foto crua a uma campanha completa feita com IA.

## Como ver localmente

Com Python instalado, na pasta do projeto:

```bash
py -m http.server 8000
```

Depois abra **http://localhost:8000** no navegador.

## Onde editar os contatos

Abra `script.js` e edite o bloco no topo (WhatsApp e Instagram). É o único lugar a mexer.

## Estrutura

- `index.html` — conteúdo da página
- `styles.css` — identidade visual e layout
- `script.js` — contatos + comparador "antes → depois"
- `assets/` — imagens e vídeos do trabalho
